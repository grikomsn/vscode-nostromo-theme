# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-01

### Added
- Initial release of Nostromo Theme
- Nostromo Theme Dark - dark teal/cyan retro sci-fi interface
- Nostromo Theme Light - light amber/cream variant
- Complete syntax highlighting for major programming languages
- Semantic token highlighting support
- GitHub Actions workflow for automated publishing
- MIT License

### Features
- Comprehensive workbench theming (sidebar, panels, terminal, status bar)
- Token colors for: comments, strings, numbers, functions, keywords, classes, types
- Terminal color scheme with ANSI colors
- Diff editor highlighting
- Debug mode theming
- Markdown preview styling
- Support for semantic highlighting
