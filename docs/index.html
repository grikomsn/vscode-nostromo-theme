NOSTROMO THEME

◈ Installation

From VSCode Marketplace

Click the button below to open the marketplace and install directly.

Install Nostromo Theme

From VSCode

Open VSCode, search "Nostromo Theme" in the Extensions panel, and click Install.

Cmd/Ctrl+Shift+X → Search "Nostromo Theme" → Install

Command Line

Use the code command to install from your terminal.

code --install-extension grikomsn.nostromo-theme

Activate the Theme

After installation, activate the theme using the Command Palette:

Cmd/Ctrl+K Cmd/Ctrl+T → Select "Nostromo Theme"
